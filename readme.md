# Start jupyter
1. Go to the Anaconda prompt.

    On Windows, press Windows key and type Anaconda prompt
    
1. Type

    ```
    jupyter notebook --notebook-dir=<folder to start juypter in
    ```
    Example:
    ```
    jupyter notebook --notebook-dir=./Downloads/pythonistics
    ```
